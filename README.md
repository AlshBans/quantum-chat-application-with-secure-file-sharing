# Quantum Random Key Generator for Secure Web Communication

## Overview
**Quantum Random Key Generator for Secure Web Communication** is a full-stack web application designed to simulate quantum-based cryptography and provide secure communication and file sharing. The system integrates **quantum computing concepts** with **cyber security principles**, allowing users to generate, control, and manage encryption keys for enhanced security, transparency, and auditability.

**Module:** Quantum-Resistant Secure File Sharing System  
**Technology Stack:** HTML, CSS, JavaScript, PHP, MySQL  

---

**System Components:**  
1. **Main System:** Quantum Random Key Generator for Secure Web Communication  
2. **Module:** Quantum-Resistant Secure File Sharing

## User-Level Workflow (Frontend)
1. **User Registration & Login**  
   - Register using email and password.  
   - Secure login with hashed password authentication.  

2. **Dashboard Access**  
   - Access: Quantum Key Generator, Secure Messaging, Secure File Sharing, Activity Logs.  

3. **Quantum Random Key Generation**  
   - Generate keys (128/256 bits) using simulated quantum randomness.  
   - Keys stored securely with metadata (creation time, usage status).  

4. **Secure Communication**  
   - Select recipient and encryption key.  
   - Encrypt messages using quantum-generated keys.  
   - Only authorized recipients can decrypt messages.  

5. **Quantum-Resistant File Sharing**  
   - Upload files and assign recipients.  
   - Encrypt files using quantum-generated keys.  
   - Apply access permissions and expiry controls.  

6. **Security Monitoring**  
   - View activity logs: login, key generation, message/file access.  

---

## Backend Workflow
- **Password Storage:** Cryptographically hashed.  
- **Quantum Random Keys:** Generated using entropy-based simulation.  
- **Encryption:** All messages and files encrypted before storage.  
- **Symmetric Cryptography with Quantum-Safe Key Management**  
- **Access Control:** Ensures only authorized users can decrypt data.  
- **Activity Logging:** For auditing and intrusion detection.
